# E-Commerce Website

A PHP-based e-commerce website with product management, admin panel, and user interface for browsing products.

## Features
- Product listing
- Admin dashboard
- Product uploads
- Responsive UI
- Database connectivity
- User-friendly interface

## Tech Stack
- PHP
- MySQL
- HTML
- CSS

## Project Structure
- `admin/` → Admin dashboard
- `css/` → Styling files
- `images/` → Website images
- `pages/` → Website pages
- `uploads/` → Uploaded products/images
- `index.php` → Main homepage

## How to Run

```bash
git clone https://github.com/krushnasrikondabattini-bit/ecommerce-website.git
```

1. Move project to XAMPP `htdocs`
2. Start Apache and MySQL
3. Import database into phpMyAdmin
4. Open:

```text
http://localhost/ecommerce-website
```
